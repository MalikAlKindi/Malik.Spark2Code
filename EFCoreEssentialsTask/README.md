# EF Core Essentials Task

This console application demonstrates:

- SQL Server database creation with `EnsureCreated()`
- Entity and `DbContext` setup
- Insert, update and delete operations
- LINQ queries with filtering, sorting and statistics
- `SaveChanges()` and tracked objects

Set `BANK_DB_CONNECTION` if your SQL Server connection string is different from the local Windows default.

```bash
dotnet restore
dotnet run
```
