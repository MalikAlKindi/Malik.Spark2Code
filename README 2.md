# C# Functions Practice

This is a small console app I made to practice C# functions.

Topics practiced:

- Built-in functions
- Math functions
- String functions
- DateTime functions
- Array functions
- Random numbers
- Type conversion
- User-defined functions
- Function overloading
- Optional and named parameters
- `ref` and `out`
- Tuples
- Recursion

Run:

```bash
dotnet run --project CsharpFunctionsPractice.csproj
```
